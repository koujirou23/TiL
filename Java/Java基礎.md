## Javaの文字出力方法
Javaはコードの最後にセミコロン(;)が必要(数値の場合""はいらない)  
```
System.out.println("Hello World"); 
```

## Javaの計算

``
System.out.println( 5 + 3 );  
``

## 文字の連結

```
System.out.println("Hello" + "World"); 
```

## 変数の定義
数値

```
int number = 3;   
System.out.println(number);
```
文字列

```
String name = "Taro";  
System.out.println(name);
```
